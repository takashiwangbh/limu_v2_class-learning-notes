# 线性代数
  
#### 标量

* 简单操作
   
* 长度
  就是绝对值

#### 向量

* 简单操作
  c = a + b where ci = ai +bi
  就是做简单操作要对每一个元素做简单操作
* 长度 
* 点乘
* 正交
  求和为0

#### 矩阵
* 简单操作
* 乘法
 矩阵乘以向量
 扭曲空间
 矩阵乘以矩阵
* 范数
 常见范数：矩阵范数，Frobenius范数

#### 特殊矩阵
* 正交矩阵
 所有行都互相正交
 所有行都有长度单位
 可以写成UUT = 1
* 置换矩阵
 置换矩阵是正交矩阵

#### 特征向量和特征值
* 特征向量是不被矩阵改变方向的向量
* 对称矩阵总是可以找到特征向量
